## О проекте

**hotel-dms** (Hotel Document Management System) — это веб-приложение для автоматизации деятельности обслуживающего персонала отеля. Система позволяет управлять договорами на услуги, вести справочники клиентов, сотрудников и услуг, распределять задачи между персоналом и формировать отчетность.

### Основные возможности

| Модуль | Функции |
|:---|:---|
| **Справочники** | Управление клиентами, сотрудниками, услугами (CRUD-операции) |
| **Договоры** | Создание, просмотр, редактирование, изменение статуса |
| **Задачи** | Назначение задач сотрудникам, отметка о выполнении |
| **Документы** | Генерация договоров в формате DOCX по шаблону |
| **Отчеты** | Формирование отчетов по услугам и занятости, экспорт в Excel/PDF |

### Цели создания

- Сокращение времени на оформление договоров и отчетности (не менее 40%)
- Повышение достоверности данных (снижение ошибок на 90%)
- Обеспечение оперативного контроля выполнения задач
- Создание единой базы клиентов, договоров и исполнителей
